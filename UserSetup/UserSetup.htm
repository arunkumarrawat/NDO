If you install NDO at your development machine, NDO is able to find the NDO Provider plug-ins â€“ it simply looks at the NDO application folder. If you install your NDO application at the users computer, the registry entry for the plug-in folder doesn't exist. This setup enables you to define a folder, which gets registered as NDO plug-in folder in the registry. The setup simply adds the following key to the registry:

 

HKEY_LOCAL_MACHINE\Software\Text & Talk GmbH\NDO\1.1

 

The value AddInPath in the key denotes the plug-in folder.
